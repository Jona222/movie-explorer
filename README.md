# 🎬 Movie Explorer

Welcome to the **Movie Explorer**! This project allows you to explore movies using TMDB API.

---

## 🚀 **Live Demo:** https://movie-explorer-seven-sigma.vercel.app/

---

## 🛠️ Tech Stac
- **Nuxt.js 3** 
- **Tailwind CSS** 
- **TMDB API** 
- **Vercel**
- **Pinia** (State Management)
